E2E & API testing met PlayWright

Oefeningen om met playwright aan de slag te gaan.

Installatie

Je begint met de initialisatie van PlayWright. Dit kun je doen door in de map PlaywrightTests een terminal(powershell) te opnenen en het volgende command in te voeren: dotnet add package Microsoft.Playwright.MSTest. Nu worden de dependencies geïnstalleerd.

De installatie is nog niet afgerond. Je mist namelijk nog de browsers voor de tests. Wel is er een powershell script aangemaakt in de bin\debug\-versie-\ map. Installeer ook de browser dependencies en gebruik hiervoor het powershell bin/Debug/net8.0/playwright.ps1 install commando.

De eerste test!

Heb je Playwright geinstalleerd; dan is het tijd voor een eerste test! Hiervoor kun je verschillende commando's uitvoerein in de terminal. Dit doe je met de normale .NET testrunner.... oftewel: dotnet test.

Commando Resultaat
dotnet test De tests worden op de achtergrond uitgevoerd.
dotnet test --settings:headed.runsettings De instellingen in de headed.runsettings file worden gebruikt.
dotnet test --settings:headed.runsettings --filter ClassName=PlaywrightTests.Oefening1 De instellingen in de headed.runsettings file worden gebruikt en de tests in de Oefening1 class worden uitgevoerd.

Testresultaten

Gebruik je de commandline? Dan zie je de resultaten van de testrun direct ook daar inclusief eventuele foutmelding. Wil je wat meer inzage? Gebruik dan de in VS ingebouwde testexplorer.
